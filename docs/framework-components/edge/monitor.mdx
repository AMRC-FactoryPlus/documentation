# Edge Monitor

The Edge Monitor monitors the status of the Edge Agents running on its cluster. It raises alerts over Sparkplug if an Edge Agent does not appear to be running correctly. It is also responsible for ensuring an Edge Agent reloads its configuration when an updated version becomes available from the ConfigDB.
